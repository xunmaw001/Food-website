# Food-website
基于Spring Boot和Vue.js的美食网站，旨在为用户提供丰富的美食信息和在线订餐服务。系统功能包括菜品浏览、食谱分享、用户点评及在线下单，创造一个美食爱好者的互动平台。后端使用Spring Boot处理数据和业务逻辑，确保高效与安全，前端采用Vue.js构建友好的用户界面，提升用户体验。该网站不仅满足用户的美食需求，还鼓励用户分享美食心得，促进餐饮文化的交流与传播。
